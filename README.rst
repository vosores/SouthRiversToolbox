How to install and uninstall?
-----------------------------

- Clone or download from https://github.com/vosores/South-Rivers-Toolbox
- Install::

	cd South-Rivers-Toolbox 
	sudo pip install pysrt

- Uninstall::
 
	sudo pip uninstall pysrt
